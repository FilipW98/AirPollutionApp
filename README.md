 ### Features:
 - This project has been made with API from: https://openweathermap.org/api
 #### Application shows:
  ##### Air pollution level:
- PM10
- PM2.5
- Co
- So2

##### Weather:
- Temperature
- Sensed temperature
- Air humidity
- Wind speed

• If You don't set any city, Wrocław will be set as default
